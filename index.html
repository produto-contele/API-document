Onde encontrar a Chave API

Siga os passos abaixo dentro do sistema Contele Teams para localizar suas credenciais:

1

Menu de Integração

No Contele Teams, acesse Configurações > Integração. Na sequência, selecione a opção "Usar API Rest". Importante: É preciso ter acesso ao Contele Teams para acessar chaves de integração.

2

Gerenciamento de Chaves

Logo abaixo você será direcionado para as orientações, podendo visualizar, copiar ou gerar novas chaves (Token e X-API-KEY).

Como testar no Swagger

O Swagger é a nossa ferramenta oficial para você visualizar e testar as requisições antes de programar.


Passo a Passo

  • Expanda o Endpoint desejado e clique em "Try it out".
  • Insira as chaves Authorization e X-API-KEY.
  • Insira os parâmetros de query obrigatórios (se houver).
  • Clique em Execute.
  • Copie o Curl gerado e aplique em sua requisição (Ex: Postman) para testar!
